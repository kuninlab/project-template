# Raw data

Good practice is to put files here that you download from somewhere else, _and then never edit them._ Record their (timestamped) provenance in this readme file.
If you need to reformat the data for use with your analysis tools and save that reformatted version, it's a good idea to (1) write a script that does that formatting, and put it in the `/scripts` folder, and (2) put the formatted files in the `data/processed` folder.
